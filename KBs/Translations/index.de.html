Dieses Werkzeug soll sich das SUMO ontology (vorgeschlagene obere vermischte Ontologie) abrufen. SUMO ist ein Starterdokument in der IEEE oberen ontologischen Standardbemühung. Zu mehr Information sehen Sie bitte http://ontology.teknowledge.com oder http://suo.ieee.org.

Berichten Sie bitte Fehler und Vorschlaegen sevcenko@vc.cvut.cz.

Sie koennen die Ontologie in einigen Weisen zugaenglich machen :
  • Schreiben Sie einen Konzeptnamen (z.B. 'Tier' oder 'angeschlossen') zum folgenden Feld. Ein entsprechendes Konzept oder eine Liste der nahen Gleicher werden angezeigt.
  • Konzept:
  • Schreiben Sie ein englisches Wort. Eine Liste der Richtungen des Wortes entsprechend Wordnet und entsprechenden SUMO Bezeichnungen wird angezeigt.
  • Englisches Wort:
  • Gehen Sie zum Baum Erbschaftsheirarchie von SUMO Konzepten.
  • Waehlen Sie eine Unterontologie von der folgenden Liste vor und zeigen Sie eine Liste von Konzepten in der Unterontologie an.
